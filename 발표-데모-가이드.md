# FactDoc — 발표·데모·심사 대응 가이드

> 발표 첫 화면 고정: **"LLM은 진실을 판단하지 않는다 — 국가 공식데이터의 룰 + 클레임그래프가 판정한다."**

---

## 1. 90초 라이브 데모 동선

1. **소비자 검증** (홈) — `"당뇨는 여주즙으로 완치된대요"`
   → **사실이 아니에요** + 사람이 읽는 설명(식약처 룰) + 📚 공식 출처 + **🧭 판정 과정 보기**(트리플 분해→룰 발화→판정 경로 시각화).
2. **통계 거짓 잡기** — `"20대 당뇨 유병률이 절반을 넘는다던데요"`
   → **허위** (KNHANES 공식 분포 대비 outlier). *수치 거짓도 공공데이터로 잡는다.*
3. **정보질문** — `"뎅기열이 뭔가요?"`
   → 판정이 아니라 **질병청 공식정보 카드**(오타 '댕기열'도 자동 교정) + 올바른 예방수칙(모기 매개).
4. **유행 → 디스커버리 퍼널** (유행 탭) — 실시간 감염병(감염병포털 실데이터) 클릭
   → 질병상세: **⚠️ 이 질병 관련 가짜정보**(클릭하면 검증) + 📘 질병청 공식정보(근거 형광펜).
5. **간판: 3분할 비교** (관리자 → 비교 데모) — 같은 주장 1건을
   → **① 무근거 LLM**(출처·판정 없음·환각) / **② 일반 RAG**(검색만, 판정 없음) / **③ FactDoc**(트리플·룰·근거체인·4단계).
6. **AI 검증 수치** (관리자 → 평가) — 아래 §3 숫자.

## 2. 공공데이터(주최기관) 사용 — 30점

- **질병관리청 국가건강정보포털**(콘텐츠 API) — 표준치료·예방 공식 의학정보 → 코퍼스·트리플·근거 형광펜
- **질병관리청 감염병포털**(data.go.kr EIDAPIService) — 전수감시 발생현황 **라이브** → 트렌드 패널·클릭 퍼널
- **질병관리청 KNHANES** — 유병률 분포 → **유병률 정합성 검사기**(통계 거짓 판정)·마이페이지 맞춤맥락
- **식품의약품안전처**(보조) — 인정기능성·부당광고 룰테이블
- → 데이터가 곧 판정 근거. 주최기관(질병청)이 코어.

## 3. AI 기술 검증 — 30점 (숫자로)

- **평가셋 133건** — 멀티에이전트 **생성↔라벨 분리 + 2인 독립 라벨 + 합의/3자 조정**(자가채점 차단), **코헨 카파 0.94**.
- **Ablation(같은 균형표본)**: **룰만 41.7% → 풀(룰+그래프) 75.0% = 클레임그래프 기여 +33.3%p**, 외부 무근거 LLM 베이스라인(**59.8%**, SNU 500건)도 상회.
- 검증코어 **100%** / 인용 정확도 / 캘리브레이션(ECE) / 과잉단정율 / 클래스별 P·R·F1.
- **환각 구조적 차단**: 인용은 LLM 생성이 아니라 클레임그래프에서 나옴 → 날조 불가. LLM 없어도(쿼터 다운) 결정론 설명으로 정확히 답.

## 4. 독창성 — 20점

- **룰 + 클레임그래프 판정**(LLM은 파싱·설명만) · **한국어 건강-클레임 온톨로지** · **식약처 법규 룰테이블**.
- **정량 모트**: 근거 트리플 **47** · 온톨로지 용어 **108** · 동의어 매핑 **412** · 커버 질환 **61** · 판정 룰 **8**.
- **5대 차별기능 전부 구현**: Why-Trace · 유병률 정합성 · Span Grounding · 근거수준 자동등급 · 커버리지 라우터(보류).
- 선행연구(ClaimsKG·GraphCheck) 명시 인용 후 선긋기: *"국가공식 의학정보를 한국어 온톨로지+식약처 룰로 결합한 첫 소비자 구현체."*

## 5. 발전 가능성 — 20점

- **GTM**: B2C(카톡 공유카드·고령층) / B2G(식약처·방심위 보완) / B2B(보험·약국·시니어케어).
- **인프라 스케일**: pgvector HNSW + tsvector + pg_trgm 인덱스로 수만 건도 풀스캔 없이. 폼팩터별 반응형 3종.
- 시장공백: SNU팩트체크 운영중단(2024.8) · 노인 유튜브 42% AI생성 가짜정보.
- 범정부 왕중왕전 진출 경로.

## 6. 심사 예상 질문 대응

- **"또 다른 RAG 챗봇 아닌가?"** → 3분할 비교로 즉시 반박. 판정은 룰+그래프, RAG는 내부 검색 부품일 뿐. ablation +33%p가 그래프 기여를 증명.
- **"정확도 100%는 자가채점 아닌가?"** → 100%는 검증코어(설계 범위)만. 광역셋은 독립 듀얼라벨(κ0.94)로 별도 채점·정직 공개. 복문은 Gemini 파서 영역(쿼터 의존).
- **"근거수준 누가 매겼나?"** → 출처 종류가 결정론으로 부여(국가건강정보포털=표준치료, KNHANES=통계, 식약처=인정기능성/반증).
- **"명예훼손 위험?"** → 사람·브랜드가 아니라 **주장만** 검증, 공식 근거 제시 후 판단은 사용자에게. 의료 면책 명시.

## 7. 가드레일 (필수)

주장만 검증(명예훼손 회피) · 환각 캐시 금지(출처·등급 플래그) · 보류를 1급 결과로 · 의료 면책 · 공공누리 4유형(비상업·변경금지) 준수(포인터+인용+요약).
