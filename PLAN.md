## 1. 최종 목표 · 한 줄 정의
- **한 줄**: *"LLM은 진실을 판단하지 않는다 — 국가 공식데이터의 룰 + 클레임그래프가 판정한다."*
- TV·홈쇼핑·유튜브·단톡방의 건강 주장을 **질병관리청(+식약처) 공식데이터**에 근거해 **사실 / 부분과장 / 허위 / 공식근거없음(보류)** 4단계로 판정하고 출처·근거체인을 보여주는 AI 서비스.
- **출품**: 2026 병무청·방위사업청·질병관리청 합동 공공데이터·AI 경진대회 — 제품·서비스 개발 부문(작동 시제품).
- **사용자/시장**: (B2C) 일반·고령 소비자 / (B2G) 식약처·방심위 모니터링 보조 / (B2B) 보험·약국·시니어케어·헬스케어 API.

## 2. 핵심 차별점 (헤드라인 — 흔들지 말 것)
1. **룰+클레임그래프 판정**(LLM은 가장자리만) → 근거체인이 보임, 환각 방지.
2. **한국어 건강-클레임 온톨로지 + 식약처 룰테이블** = 못 베끼는 모트.
3. **"RAG"는 헤드라인에서 제거**(내부 부품명만). "또 RAG"로 1초 분류되는 게 최대 실점 리스크.
4. **보류를 1급 결과로**(공식근거 없으면 억지 판정 안 함) = 정직성=신뢰.

## 4. 3대 전략 방향 + 권고

| 방향 | 점수 강점 | 현재 | 권고 |
|---|---|---|---|
| **S1. 프리벙킹 조기경보 레이더** | 독창성20·AI30 | 급증신호 보유 | **★헤드라인 신규 — 1순위.** "사후검증→사전예방, 예측형 관제"로 독창성 차별화. 감염병 급증 + (조달 발주·검색트렌드) 선행지표 → 가짜뉴스 선제검증 + 관리자 카드뉴스/해명자료 초안 자동생성 |
| **S2. 병무청×질병청 군 융합** | 공공데이터30(만점 lever) | 또래 비교 부분구현 | **저비용 강화 — 2순위.** 군 밀집시설 감염병+괴담 탭. 주최기관 2개(병무+질병) 동시 사용 = 공공데이터 만점 서사 |

## 5. 백로그 (이번 세션 요청 — 빠짐없이) 
> 상태: ⬜예정 / 🔄진행 / ✅완료. 디테일·주의까지 캡처(요청이 또 누락되지 않도록).

### 5-A. 버그·즉시수정
- ✅ **불만족 삭제 시 카운트 미감소 버그** → 삭제 시 down 감소(floor 0). + **만족/불만족 카운트 초기화** 버튼(패널·전체화면 모달).
- ✅ **부실응답 큐 전체화면** 펼쳐보기 관리(체크박스·CSV/TXT·필터·페이지네이션). *(관리자→대시보드→부실응답 큐→"⛶ 전체화면 검토")*

### 5-B. 감염병 현황판 — 추이 그래프
- ⬜ **칩별 표시 단위 정합**: 6개월 칩 = **주(週)단위**(현재 일단위라 과밀) · 1년 = **월단위**(이미) · 1주=일 · 1개월=일 · 3년=월/분기. (전국 기준 EpiTrend의 `WIN_NAT`·`tg` 로직 조정)
- ⬜ **추이 포인트 클릭 → 그 시점 상위 발생 감염병 Top5** 표시(전체 감염병 그래프). 호버=값, **클릭=그 주/월의 상위 5개 감염병 + 건수**(EID 주별/월별 per-disease 데이터로 산출, 신규 API 불필요).

### 5-C. 인구밀집도 기반 확산률
- ⬜ **시도별 인구밀도(인구/면적) × 발생률 상관 시각화** — "밀집도 높을수록 전파 빠름". 17개 시도 면적은 고정값 하드코딩 + 기존 인구추정(`sidoPop`) → 밀도 산출(신규 API 없이 1차 가능). 정밀화 시 SGIS `ppltn_dnsty`. 산점도(밀도 vs 발생률) + 밀도보정 확산위험지수.

### 5-D. 병무 데이터 — 연도별·나이대별(좁은 또래)
- ⬜ **현재 단일 평균(2024 174.54cm/73.27kg)을 → 연도별 + 만나이별 평균으로 확장.**
  - **이유(사용자)**: 10대는 성장기라 10살↔19살 키·몸무게 격차 큼. 20대도 ±1세에 평균이 달라짐 → "또래"를 **±1~2세로 좁게**.
  - 병무청 병역판정검사 신체검사 통계(연도별·연령별 평균 키/몸무게/BMI) 수집(있는 범위). 마이페이지에서 입력 나이 ±좁은 밴드로 비교. 데이터 없으면 솔직히 표기.

### 5-E. 미상 효능 답변 보강
- ⬜ **말벌주·민간 보양주/보양식 등 코퍼스 미수록 항목도 답변.** 보류(1급) + 성분/카테고리 안내 + **주류=알코올 안전경고**(과음 위험). 단정 회피·의료면책. ("효능 입증 공식근거 없음 + 술이라 건강상 주의" 식)

### 5-F. 프리벙킹 레이더 (S1)
- ⬜ 관리자 대시보드 패널: 급증 감염병 → "2주 내 관련 가짜뉴스 검색 급증 예상" 조기경보. + **카드뉴스/해명자료 초안 자동생성**(질병청 공식 증상·예방 인용). (외부 선행지표=조달 발주/검색트렌드는 배치 캐시, 점수 무관 보조)

### 5-G. 추가 API (사용자 활용신청용)
- ✅ 조사 완료 → [데이터-API-조사.md](데이터-API-조사.md). 우선순위: ①식약처 부당광고(15058599) ②인구밀도(SGIS/하드코딩) ③조달 발주계획(15129462)+입찰(15129394) ④네이버 데이터랩.

## 6. 창의 기능 아이디어 ("반짝이는" — 식상하지 않게)
- 💡 **국가 조달이 보내는 신호**: 정부가 마스크·진단키트·해열제를 대량 발주하면 = "곧 뭔가 온다". 조달 데이터를 유행/인포데믹 선행지표로 → "검색·챗봇·뉴스가 모르는, 국가 행정의 선행 신호"(독창성 한 방).
- 💡 **프리벙킹 카드 자동생성**: 가짜뉴스가 뜨기 전, 질병청 공식근거로 만든 "예방접종 카드"를 담당자가 원클릭 배포.
- 💡 **신뢰도 캘리브레이션 곡선 공개**: "우리 신뢰도 80%는 실제 80% 맞다"를 reliability diagram으로 → AI 검증 점수의 정직성 증명.
- 💡 **개인화 치명도 지수**(S3): "이 가짜뉴스, 당신(70대·당뇨)에겐 KNHANES 기준 합병증 상위 10% 위험."
- 💡 **어르신 모드**: 큰 글씨·음성 읽어주기·카톡 공유카드(이미 일부) — 표적(고령) 직격.
- 💡 **가짜뉴스 계보 추적**: 같은 주장의 변종("여주즙"→"여주환"→"여주분말")을 온톨로지로 묶어 한 번에 방어.

## 7. 데이터 소스
- **코어(주최기관 충족)**: 질병청 국가건강정보포털(코퍼스)·감염병포털 EDW(현황판)·KNHANES(개인맥락)·만성질환통계.
- **보조**: 식약처 인정기능성(반증)·**식약처 부당광고(15058599, 신규 권장)**·병무청 신체검사 통계(또래).
- **신규 후보**: 인구밀도(SGIS/하드코딩)·조달청 18종(방역물자 선행지표)·네이버 데이터랩(검색트렌드, 민간).
- 상세: [데이터-API-조사.md](데이터-API-조사.md). 원칙: **per-request 외부호출 금지 → 배치 캐시**(§13.7). 공공누리 4유형(출처표시·비상업·변경금지) 준수: 짧은 인용+출처+자체요약.

## 8. 아키텍처 (요약)
추출(작은 LLM)→시맨틱 캐시→하이브리드 검색→**룰+클레임그래프 판정**→근거체인+신뢰도→캐시. 스택: Vite+React+TS+Tailwind / Supabase(pgvector) / Cloudflare Pages / Supabase Edge Functions(판정·LLM) / BGE-m3 / Gemini Flash(가장자리) / GitHub Actions cron.

## 9. 평가 기준 매핑
- 공공데이터30 = 질병청 코어 + 식약처 + 병무청(3 주최기관) + 조달/인구밀도(폭). 
- AI30 = 판정엔진 + 라벨셋 정확도·인용정확도·환각률·캘리브레이션 + 무근거LLM vs 우리 ablation.
- 독창성20 = 클레임그래프·온톨로지·룰판정 + **프리벙킹 예측 관제** + 조달 선행지표.
- 발전가능성20 = B2C/B2G/B2B GTM + 개인화 치명도 + 범정부 진출.
