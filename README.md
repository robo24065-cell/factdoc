# FactDoc — 국가 공식데이터 기반 건강정보 팩트체커

> **LLM은 진실을 판단하지 않습니다 — 국가 공식데이터의 룰과 클레임그래프가 판정합니다.**

TV·홈쇼핑·유튜브·단톡방에 퍼지는 건강 주장을, 질병관리청(+식품의약품안전처)의 **국가 공식데이터**에 근거해
**사실 / 부분적·과장 / 근거없음·허위 / 공식근거없음·보류** 4단계로 판정하고, 출처와 근거 체인을 함께 보여주는 AI 서비스.

2026 병무청·방위사업청·질병관리청 합동 공공데이터·AI 활용 경진대회 — 제품 및 서비스 개발 부문 출품작.

---

## 핵심 차별점 (왜 "또 다른 RAG 챗봇"이 아닌가)

1. **판정은 LLM이 하지 않는다.** 지저분한 사용자 문장을 LLM(Gemini)이 **트리플(주체·관계·대상질환)로 파싱**하고 설명문만 생성한다.
   진실 판정은 **결정론적 룰 + 클레임그래프 트리플 매칭**이 한다 → 환각 방지 + "왜 그렇게 판정했나"가 보인다.
2. **근거수준은 출처가 결정한다.** 국가건강정보포털(표준치료)·KNHANES(통계)·식약처(인정기능성/부당광고)별로 근거수준이 자동 부여된다.
3. **Why-Trace.** 주장 분해 → 룰/그래프 발화 → 판정 경로를 시각화 (설명가능성).
4. **검증 티어.** 자동 판정은 `자동·미검증`, 사람 검토분만 `검증완료` 승격 → 정직성과 확장성을 동시에.
5. **Gemini가 없어도 동작.** 규칙 파서 + 결정론 설명문으로 LLM 다운/쿼터소진에도 정확한 판정과 사람이 읽을 답을 제공.

## 4단계 판정 루브릭

| 판정 | 의미 |
|---|---|
| **사실** | 핵심 트리플이 공식 근거와 방향·강도·한정조건 모두 일치 |
| **부분적·과장** | 방향은 맞으나 강도·범위 과장 / 한정조건 누락 / 인과 비약 |
| **근거없음·허위** | 공식 근거와 정면 모순 / 비가역 만성질환 완치 표방 / 식품·건기식의 질병 치료·예방 표방(식약처 룰) / 식품 섭취로 감염 |
| **공식근거없음·보류** | 대조할 공식 근거가 코퍼스에 없음 (1급 결과 — 억지 판정 안 함) |

## 데이터 (주최기관 = 질병관리청 코어)

- **질병관리청 국가건강정보포털** — 질병별 표준치료·예방 공식 의학정보 (콘텐츠 API 적재 → 청크·트리플)
- **질병관리청 감염병포털** (data.go.kr EIDAPIService) — 전수감시 발생현황 라이브(트렌드 패널·클릭 팩트체크 퍼널)
- **질병관리청 KNHANES** — 통계 근거 / 마이페이지 맞춤맥락
- **식품의약품안전처** — 건강기능식품 인정기능성 + 부당광고 룰테이블(보조 모듈)

## 평가 (AI 기술 검증)

- **골드 평가셋**: 멀티에이전트로 구축 — 생성↔라벨 분리 + **2인 독립 라벨러 + 합의/3자 조정**(자가채점 차단), **코헨 카파 0.94**.
- **지표**: 4클래스 정확도 · 클래스별 P/R/F1 · 매크로F1 · 인용 정확도 · 캘리브레이션(ECE) · 과잉단정율 · 보류 적정성 · 티어별.
- **Ablation**: 무근거 LLM / 일반 RAG / 룰만 / 풀(룰+그래프) 비교 — 외부 기준선(한국어 무근거 LLM 59.8%, SNU 500건) 병기.
- 관리자 화면 `/admin/eval`에서 라이브 채점.

## 아키텍처 / 스택

```
사용자 주장 → ① 의도 분류(검증 vs 정보질문)
            → ② Gemini 파싱(트리플) + 규칙 파서 결합
            → ③ 시맨틱 캐시(임베딩 ANN + 규칙 호환 가드) · 하이브리드 검색(벡터+tsvector RRF)
            → ④ 룰 + 클레임그래프 판정 → 4단계 + 근거체인 + Why-Trace
            → ⑤ 결정론/Gemini 설명문 → 캐시(버전 게이팅)
```

- **프론트**: Vite + React + TS + Tailwind v4 · **DB/벡터/검색**: Supabase(Postgres + pgvector HNSW + tsvector + pg_trgm)
- **판정/LLM 오케스트레이션**: Supabase Edge Functions(Deno) · **임베딩/파싱/설명**: Gemini(가장자리만) · **호스팅**: Cloudflare Pages

## 안전·법무 가드레일

- **명예훼손 회피**: 사람·브랜드·프로그램이 아니라 **주장(claim)만** 검증한다.
- **환각 방지**: 자동 판정은 출처·신뢰도·등급(자동·미검증) 플래그. 인용은 그래프에서 나오므로 날조 불가.
- **의료 면책**: 진단·치료 대체가 아님. 증상 의심 시 전문가 상담 안내.
- **저작권**: 국가건강정보포털 공공누리 제4유형(비상업·변경금지) — 원문 통째 재배포 금지, 출처 포인터+짧은 인용+자체 요약+추출 메타데이터만.
